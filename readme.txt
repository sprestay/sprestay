1) Умный сервис прогноза погоды (Задачка со звездочкой)
2) AJAX - так как функционал сайта крайне небольшой, перезагружать целиком сайт нет нужды
    Python - простой синтаксис, готовые библиотеки
    Реализован сайт на Python Flask
    Ответ сервера - JSON, который возвращают Api для геолокации и прогнозов погоды.
    Отрисовка ответа происходит с помощью JS на стороне клиента. Пользователь получает совет в виде текста

3) Яндекс диск https://yadi.sk/i/vbWzIBkjITulAw
4) Пользователь заходит на сайт и вводит название города
    JavaScript проверяет что город введен, и введен верно
    Затем отправляется AJAX запрос на сервер
    Python с помощью api yandex geocoder получает координаты населенного пункта
    Затем по координатам получаем прогноз погоды с помощью api яндекс.погода
    Прогоняем полученный прогноз через фильтр условиями - получаем строку с рекомендацией
    Возвращаем клиенту JSON прогноза погоды и нашу строку рекомендации
    JavaScript отрисовывает JSON на клиенте
